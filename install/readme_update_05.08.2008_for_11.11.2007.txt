Перед установкой сделайте следующее:

1. Завершите работу всех программ СГТС, запущенных на клиентах

2. Удалите предыдущую версию СГТС без SQL-скриптов 
(через панель управления или через главное меню)

3. Очистите папку, в которую была установлена версия sgts_oracle_01.08.2007.exe
(по умолчанию c:\Program Files\СГТС)

4. Убедитесь что на жестком диске, на который будете устанавливать 
достаточно места. Необходимо 200 Mb.

5. Установить новую версию из файла: sgts_oracle_update_05.08.2008_for_11.11.2007.exe
(SQL-скрипты необходимо ставить только 1 раз, все остальные установки без SQL-скриптов)

Установка длится около 7 часов.
